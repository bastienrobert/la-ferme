# La Ferme - Server

This package contains the server application using GraphQL and PostgreSQL.

## Install

Check **INSTALL.md**

## Getting started

Use **yarn** as package manager.

Start postgresql:

```
brew services start postgresql
```

Then:

```
yarn
yarn dev
```
